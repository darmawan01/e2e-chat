### Peer to Peer Simple Chat APP With Electron JS

## App 1 (app-1)
The client side chat App
### Spin Up
- Go to the app-1
```
cd app-1
```

- Install the dependencies
```
yarn

OR

npm i
```

- Run the app
```
yarn start
```
